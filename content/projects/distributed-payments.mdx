This project explores how to design **resilient payment orchestration** systems.

## Architecture

The system uses an **event-driven microservice architecture**.

Core components:

- API Gateway
- Payment Orchestrator
- Event Bus (Kafka)
- Fraud Detection Service

## Reliability Strategy

Key techniques:

- idempotent operations
- circuit breakers
- retry queues
- distributed tracing

## Lessons Learned

Simplicity matters more than perfect abstraction.
